How to check Database mode?
  • Step 1: Open the SQL server management studio (SSMS) in Wd's system
  • Step 2: Click on the Drop down icon and check whether Minerva is in Normal mode/Corrupted/Suspect/single user/recovery mode
  • Step 3: Click on the Drop down icon and select the latest Minerva (eg: Minerva 2022)
  • Step 4: Execute the Query (DBCC CHECKDB)
FSU sequence - in sequence/not in sequence
  • Step 1: Go to the SQL server management studio (SSMS) in Wd's system → Click on New query → Execute the FSU Sequence query.
  • Step 2: If the FSU's are in correct Sequence, then the query will be executed successfully.
  • If NOT the query will be executed with the errors
    • Step 1: We need to check how many rows has been affected in the Execution result part.
    • Step 2: Need to share those details to our Development team.
    • Step 3: After the completion for the FSU RESET process from our Development team.
    • Step 4: Need to login the Forum MicroERP, so that the FSU will starts installing.
    • Step 5: According to the numbers of the FSU's, the installation time will be taken.
    • Step 6: Once after the FSU Installation gets completed and the forum gets logged in.
    • Step 7: We need to logout the Forum and open the SQL server management studio (SSMS).
    • Step 8: Again we need to cross verify whether the FSU is in correct sequence or not.
    • Step 9: After the executing the FSU query, we come to know that the FSU are in correct Sequence.
How to find the status Anti-virus and Window Firewall?
  • Step 1: Go to Windows security-- click on the "FireWall and network settings" and check whether it is turned on/off.
  • Step 2: Check any third party anti-virus is installed or not (eg : Quick heal , Avast , MacAfee ) and disable it.
ForumApp running in the component services - Yes/No

Search for Component services → Click on Computer à My Computer à “COM+ APPLICATION” → Right click on "ForumAPP" and click "Start".
If the ForumApp is running, then it is in good state.


IF NOT, THE FORUMAPP WAS NOT RUNNING IN THE COMPONENT SERVICES

Delete that existing ForumApp in the component services and need to create a new one.
To create a new ForumApp in component services.

  • Step 1: Click on back menu and go to the Main Component services page.
  • Step 2: Click on Computer à My Computer à Right click on “COM+ APPLICATION” à “Database”àthen select "NEW DATABASE" create a new application and name it as "ForumApp".
  • Step 3: To build the connection → Click on Local disk C → Windows → Syswow64 → CTL → decrypt encrypt.dll
  • Step 4: After the connection built → Right click on "ForumAPP" and click "Start" then the ForumApp will run fortunately.
