Not updated in Tenovos - Title
Category : ISCA-Asset listing
Sub-Category : Update Title
Priority : Medium
Type : INC
Inputs :
  1. User ID(type of user)
  2. Asset Name/Asset URL
  3. Title of asset to be updated.
  4. URL of asset in Tenovos
  5. Screenshots
Diagnosis :
  1. Verify the user's privilege to edit the asset.
  2. Verify the issue whether it is appropriate.
  3. Verify the asset with name and compare with screen shot
  4. Verify the user browser
  5. Verify user internet connection stability.
Troubleshooting :
  • Due to a lack of edit permissions, the user was unable to modify the asset's title.
  • If the user wishes to edit assets, please request an update to their user role.
  • Access the asset through the Content Portal and modify the title.
Resolution : To update the asset title:
  1. Login to ISCA
  2. Select the asset which need to update
  3. Update the title by clicking on the “Edit” button beside the “Basic Asset Information” Section(marked in box):
  4. After editing the title, Click on “Save” button.
  5. After successful updating of title, Please check “Store Front” option and click on “Save” button, then it will update in tenovos.

Category : Content Portal
Sub-Category : Content Ingestion
Priority : Medium
Type : INC
Inputs :
  1. Screenshot of the error
  2. USER ID (type of user)
  3. Ingestion Title

  1. Verify that an ingestion with the same content has not been created previously. If a duplicate ingestion exists, you cannot create a new one.
  2. If the user does not have access, the following warning will be popped up:
  3. Enquire with the user if they possess the necessary privileges to initiate the ingestion process through manual upload.

  1. check with user about the format they are attempting to upload the asset in.

Resolution :
  • User should have necessary permission to create an ingestion manually.
  • The Asset should be in a prescribed format for uploading.
  • To create ingestion through manual upload you need to follow these steps:
    Click on new ingestion
  • Select the desktop/ device from source path to upload the images manually.
