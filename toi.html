MS Popup not opening
Category : ISCA-Assets
Sub-Category : AD Login
Priority : Medium
Type : INC
Inputs :
  1. User ID (type of user)
  2. Screenshots
  3. Connected to NetSkope Client
  4. Browser Permissions for pop-up notification
  5. Stability of Internet Connection.
Diagnosis :
  1. Check for user access with VPN (NetSkope Client)
  2. Check for the user role.
  3. Check for the user internet connection stability.
  4. Ensure that user is using a supported and up-to-date web browser.
  5. Check user security software, including antivirus or firewall, blocking the pop-up.
Troubleshooting :
Resolution : Pop-up working
